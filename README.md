# 🎮 Jogo da Memória

Um jogo da memória simples, divertido e interativo, feito para web.  
O objetivo é encontrar todos os pares de cartas no menor tempo possível.  

---

## 🛠 Tecnologias

- HTML5  
- CSS3  
- JavaScript (ES6)

---

## 🚀 Como jogar

1. Abra o `index.html` no navegador.  
2. Clique nas cartas para revelar os pares.  
3. Combine todos os pares para ganhar!  
4. Use o botão **RESET GAME** para reiniciar.
